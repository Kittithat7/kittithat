Data Type in Python

Python คืออะไร

Python เป็นภาษาโปรแกรมที่มีความนิยมสูงและใช้กันอย่างแพร่หลาย โดยมันได้รับการออกแบบให้เข้าใจง่าย และมีโครงสร้างที่ชัดเจน อ่านง่าย เหมาะสำหรับผู้เริ่มต้นในการเรียนรู้การเขียนโปรแกรมและยังสามารถใช้ในการพัฒนาโปรแกรมที่ซับซ้อนได้
เราจะมาเรียนรู้ประเภทข้อมูลหรือเรียกกันว่า Data type แบบง่ายๆกัน เพียงแค่ กดที่นี่ เพื่อ Compiler code บนเบราว์เซอร์

ประเภทข้อมูล str (String)

ประเภทข้อมูล str คือประเภทข้อมูลที่ใช้สำหรับเก็บข้อความ ซึ่งประกอบด้วยตัวอักษรหรือสัญลักษณ์ใดๆ ที่ถูกจัดเก็บไว้ในรูปแบบของลำดับของตัวอักษร โดยข้อความใน str จะอยู่ภายใน ('.........') หรือ (".........")
เช่น "Hello Python"

ตัวอย่าง: text = "สวัสดี, Python!"
print(text) # Output: สวัสดี, Python!

ประเภทข้อมูล int (Integer)

ประเภทข้อมูล int ใช้เก็บตัวเลขจำนวนเต็ม เช่น 5, -7 หรือ 100

ตัวอย่าง: a = 5
b = 6
print(a + b) # Output: 11

ประเภทข้อมูล float (Decimal Numbers)

ประเภทข้อมูล float ใช้เก็บตัวเลขที่มีจุดทศนิยม เช่น 3.14 หรือ 7.5

ตัวอย่าง: a = 11.5
b = 6.3
print(a + b) # Output: 17.8

ประเภทข้อมูล complex (Complex Numbers)

ประเภทข้อมูล complex ใช้เก็บจำนวนเชิงซ้อนที่มีส่วนจริงและส่วนจินตภาพจะมี J หรือ j ต่อท้ายค่านั้น

ตัวอย่าง: x = 3 + 4j
print("Real part:", x.real) # Output: 3.0
print("Imaginary part:", x.imag) # Output: 4.0

ประเภทข้อมูล list (List)

ประเภทข้อมูล list คือชุดของข้อมูลที่สามารถเก็บหลายๆ ค่าได้ภายในตัวมันเอง ซึ่งสามารถมีทั้งตัวเลขและข้อความได้

ตัวอย่าง: fruits = ["apple", "banana", "cherry"]
fruits.append("orange")
print(fruits) # Output: ['apple', 'banana', 'cherry', 'orange']

ประเภทข้อมูล tuple (Tuple)

ประเภทข้อมูล tuple คล้ายกับ list แต่ไม่สามารถแก้ไขค่าใน tuple ได้หลังจากสร้างขึ้น

ตัวอย่าง: fruits = ("apple", "banana", "cherry")
print(fruits[0]) # Output: apple

ประเภทข้อมูล set (Set)

ประเภทข้อมูล set ใช้เก็บข้อมูลที่ไม่ซ้ำกันและไม่สนใจลำดับของข้อมูล

ตัวอย่าง: fruits = {"apple", "banana", "cherry"}
fruits.add("orange")
print(fruits) # Output: {'apple', 'banana', 'cherry', 'orange'}

ประเภทข้อมูล dict (Dictionary)

ประเภทข้อมูล dict ใช้เก็บข้อมูลที่เป็นคู่ของคีย์และค่า เช่น {"name": "John", "age": 30}

ตัวอย่าง: person = {"name": "John", "age": 30}
print(person["name"]) # Output: John
print(person["age"]) # Output: 30

ประเภทข้อมูล boolean (Boolean)

ประเภทข้อมูล boolean ใช้เก็บค่าที่เป็น True หรือ False เท่านั้น

ตัวอย่าง: x = 7
y = 10
result = x < y # True
print(result) # Output: True
